## Base de donnée
Une **base de données** est un système organisé qui permet de stocker, gérer et récupérer des données de manière efficace. Elle est conçue pour faciliter l'accès et la manipulation des informations, garantissant ainsi leur intégrité et leur sécurité. Les bases de données peuvent être classées en plusieurs types, notamment les bases de données relationnelles, qui utilisent des tables pour structurer les données, et les bases de données NoSQL, qui offrent une flexibilité accrue pour le stockage de données non structurées.
