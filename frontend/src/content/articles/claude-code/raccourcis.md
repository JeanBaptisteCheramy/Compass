## Changement de mode
 ### shift + tab
 
   - **_plan mode_** : mode de planification, utile à l'initialisation du projet
   - **_accept edits on_** : ne demandera pas la permission pour les modifications de fichiers (sauf permissions lourdes)
   - **_?_** : c'est le mode prompt, il attend les instructions
