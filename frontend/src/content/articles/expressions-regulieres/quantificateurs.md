## Quantificateurs

### Zéro ou plusieurs `*`

``` regexp
/xy*z/
```

### Zéro ou un `?`

``` regexp
/colou?r/
```

### Un ou plusieurs `+`

``` regexp
/xy+z/
```

### Exactement n `{n}`

``` regexp
/nn/
```

### Au moins n `{n,}`

``` regexp
/a{2,}/
```

### Entre n et m `{n,m}`

``` regexp
/a{2,4}/
```
