## L’équilibre « Qualité – Coût – Délais »

https://www.visual-planning.com/fr/wp-content/uploads/2022/06/triangle-dor.png

### Le triangle de projet

Un projet repose sur trois contraintes principales :

- **Qualité** : capacité à répondre aux besoins
- **Coût** : ressources financières nécessaires
- **Délais** : temps de réalisation

Modifier un élément impacte les autres.

Exemples :
- Réduire le délai → augmente souvent le coût
- Réduire le coût → peut diminuer la qualité

Chaque projet doit définir :
- ce qui est **fixe**
- ce qui est **négociable**
